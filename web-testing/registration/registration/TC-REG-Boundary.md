# Граничные тест-кейсы — Регистрация пользователя

TC-REG-BV-001 — Минимально допустимая длина пароля

Тип: Boundary

Предусловия: Пользователь не зарегистрирован

Тестовые данные:

Пароль: 8 символов (минимум)

Пример: 12345678

Шаги:

Заполнить все поля валидными данными

Ввести пароль из 8 символов

Подтвердить пароль

Поставить галочку в чекбоксе

Кликнуть «Зарегистрироваться»

Ожидаемый результат:
Регистрация успешно выполняется.

TC-REG-BV-002 — Пароль меньше минимальной длины

Тип: Boundary / Negative

Тестовые данные:

Пароль: 7 символов

Пример: 1234567

Шаги:

Заполнить все поля валидными данными

Ввести пароль из 7 символов

Подтвердить пароль

Кликнуть «Зарегистрироваться»

Ожидаемый результат:
Отображается ошибка «Пароль слишком короткий», регистрация не выполняется.
