# TC-REG-002 — Регистрация с пустыми обязательными полями.
# Предусловия: Перейти на сайт https://demo1wp.shoporg.ru/

Шаги:

1. Открыть страницу регистрации

2. Оставить обязательные поля пустыми

3. Нажать «Зарегистрироваться»

Ожидаемый результат:
Отображаются сообщения о необходимости заполнения полей.

# TC-REG-003 — Регистрация с невалидным email.

Шаги:

1. Ввести email в формате test@com

2. Заполнить остальные поля валидно

3. Нажать «Зарегистрироваться»

Ожидаемый результат:
Отображается ошибка «Некорректный формат email».

TC-REG-004 — Регистрация с несовпадающими паролями.

Шаги:

Ввести пароль

Ввести другой пароль в поле подтверждения

Нажать «Зарегистрироваться»

Ожидаемый результат:
Отображается ошибка «Пароли не совпадают».
