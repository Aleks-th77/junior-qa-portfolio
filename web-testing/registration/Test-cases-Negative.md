## TC-REG-002 — Регистрация без заполнения обязательных полей
# # Предусловия: Перейти на сайт https://vseinet.ru/registr/

Шаги:
1. Оставить обязательные поля пустыми
2. Кликнуть кнопку «Зарегистрироваться»

Ожидаемый результат:
- Отображаются ошибки валидации
- Регистрация не выполняется


## TC-REG-003 — Регистрация с невалидным email

Тестовые данные:
- Email: Test@com

Шаги:
1. Заполнить все поля валидными данными
2. Ввести email в формате `Test@com`
3. Поставить галочку в чекбоксе
4. Кликнуть кнопку «Зарегистрироваться»

Ожидаемый результат:
- Отображается сообщение «Некорректный формат email»
- Регистрация не выполняется

## TC-REG-004 — Пароль и подтверждение не совпадают

Шаги:
1. Заполнить все поля валидно
2. Ввести пароль `123456789`
3. Ввести подтверждение пароля `12345678`
4. Кликнуть кнопку «Зарегистрироваться»

Ожидаемый результат:
- Отображается сообщение «Пароли не совпадают»
- Регистрация не выполняется

## TC-REG-005 — Регистрация без согласия с условиями

Шаги:
1. Заполнить все поля валидными данными
2. НЕ ставить галочку в чекбоксе
3. Кликнуть кнопку «Зарегистрироваться»

Ожидаемый результат:
- Кнопка недоступна ИЛИ отображается ошибка
- Регистрация не выполняется
  
## TC-REG-006 — Регистрация с невалидной датой рождения

Шаги:
1. Ввести дату рождения в будущем
2. Заполнить остальные поля валидно
3. Кликнуть кнопку «Зарегистрироваться»

Ожидаемый результат:
- Отображается сообщение об ошибке даты
- Регистрация не выполняется
