# Регистрация пользователя
# TC-REG-001 — Регистрация с валидными данными.
Предусловия: Перейти на сайт https://demo1wp.shoporg.ru/, Пользователь не зарегистрирован

Шаги:

1. Открыть страницу регистрации

2. Ввести валидное имя

3. Ввести валидный email

4. Ввести валидный номер телефона

5. Ввести валидный пароль

6. Подтвердить пароль

7. Нажать «Зарегистрироваться»

Ожидаемый результат:
Пользователь успешно зарегистрирован, отображается сообщение об успехе.
