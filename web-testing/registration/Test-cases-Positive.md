# Регистрация пользователя
# TC-REG-001 — Регистрация с валидными данными.
Предусловия: Пользователь не зарегистрирован

Шаги:

Открыть страницу регистрации

Ввести валидное имя

Ввести валидный email

Ввести валидный номер телефона

Ввести валидный пароль

Подтвердить пароль

Нажать «Зарегистрироваться»

Ожидаемый результат:
Пользователь успешно зарегистрирован, отображается сообщение об успехе.
