# TC-DEL-002 — Отмена удаления аккаунта.

Шаги:

1. Нажать «Удалить аккаунт»

2. Нажать «Отмена»

Ожидаемый результат:
Аккаунт не удалён, пользователь остаётся авторизован.

# TC-DEL-003 — Удаление аккаунта с неверным паролем.

Шаги:

1. Нажать «Удалить аккаунт»

2. Ввести неверный пароль

3. Подтвердить

Ожидаемый результат:
Отображается ошибка, аккаунт не удалён.
