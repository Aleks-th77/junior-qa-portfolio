# Восстановление пароля.
# TC-REC-001 — Восстановление пароля по телефону.
Предусловия: К аккаунту привязан номер телефона

Шаги:

1. Нажать «Забыли пароль?»

2. Выбрать восстановление по телефону

3. Ввести корректный номер

4. Ввести код из SMS

5. Установить новый пароль

Ожидаемый результат:
Пароль успешно изменён.
